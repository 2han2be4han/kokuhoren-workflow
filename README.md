# 国保連請求ワークフロー

放課後等デイサービス／児童発達支援の国保連請求フローを、工程ごとにチェック・進捗管理するためのWebアプリです。聴覚障害のある職員も使えるよう、色・アイコン・短い説明文だけで判断できるUIにしています。

---

## できること

- **11フェーズ＋差戻対応**を、添付テキストの作業項目どおりにチェック管理
- 各項目に **未完了 ○ ／ 作業中 ▶ ／ 完了 ✓** の3状態（色＋アイコン＋文字で判別）
- **重要期限（3日・4日・5日）** を画面上部に常時表示。残り日数を大きく表示
- 同じ書類は **全フェーズで同じ色・同じアイコン・太字** で表示
- **⑧-1明細作成** は登録した児童ごとに項目を自動生成（一人＝通常処理／兄弟＝上限管理手順アコーディオン展開・各ステップもチェック可）
- **差戻フロー** は冒頭で差戻対象児童（自社・他社）を選択 → YES/NOで連動項目が自動で開き、選んだ児童を各項目で消し込み。兄弟（上限管理）が含まれるときだけ上限管理FAX項目が開く
- **施設・対象管理タブ** で自社児童（兄弟チェック付）・他社児童・上限管理FAX先を追加／削除
- **進捗の自動保存**（端末内＋Googleスプレッドシート）
- **途中再開**：最後の「作業中」項目へ自動スクロール、任意項目へジャンプ
- スマホ・PC両対応

---

## ローカルで確認する

```bash
cd kokuhoren-workflow
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

この時点では **端末内保存（localStorage）だけで全機能が動きます**。スプレッドシート連携は任意です。

---

## Googleスプレッドシート連携（任意）

1. Googleスプレッドシートを新規作成
2. 拡張機能 → Apps Script を開く
3. `gas/code.gs` の中身を貼り付けて保存
4. デプロイ → 新しいデプロイ → 種類「ウェブアプリ」
   - 実行ユーザー：自分
   - アクセスできるユーザー：全員
5. 発行された **URL（/exec で終わる）** をコピー
6. アプリの「⚙️ スプレッドシート設定」に貼り付けて「保存して連携開始」

以降、チェックのたびに自動でスプレッドシートへ保存されます（画面はそのまま・再読込なし）。別端末や翌月の途中再開は「クラウドから読込」で復元できます。

### 作られるシート
| シート | 内容 |
|---|---|
| 進捗ログ | 項目単位の状態（対象月・項目ID・状態・更新日時） |
| 施設マスタ | 自社児童（兄弟フラグ）・他社児童・上限管理FAX先 |
| 月次サマリ | 月ごとの完了数・進捗率 |

## GitHub Pagesで公開する

1. このフォルダをGitHubリポジトリにpush
2. Settings → Pages → Branch を `main` / `(root)` に設定
3. 発行されたURLにアクセス

`index.html` と `workflow-data.js` だけで動作します（`gas/` はGAS用の参考ファイルで、公開には不要です）。

---

## ファイル構成

```
kokuhoren-workflow/
├── index.html          メイン（CSS・JS内包）
├── workflow-data.js    全工程・書類・期限のデータ定義
├── gas/code.gs         GAS側スクリプト（連携時のみ使用）
└── README.md
```

工程や書類を変更したいときは `workflow-data.js` を編集してください。
